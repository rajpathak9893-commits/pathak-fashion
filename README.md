# Pathak Fashion 🛍️

An E-commerce website built using Django.

## 🚀 Features
- User Registration & Login
- Product Listing
- Add to Cart
- Checkout System
- Order Management
- Admin Panel

## 🛠️ Technologies Used
- Python
- Django
- HTML, CSS
- SQLite

## ▶️ How to Run
1. git clone https://github.com/rajpathak9893-commits/pathak-fashion.git
2. cd pathak-fashion
3. python manage.py runserver
